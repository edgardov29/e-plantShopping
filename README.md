# e-Plant Shopping 🌱

e-Plant Shopping es una pequeña tienda online de plantas de interior donde el usuario puede explorar diferentes categorías de plantas, agregarlas al carrito y ver el detalle de su compra.

## 🚀 Demo

Puedes ver la versión desplegada aquí:  
https://edgardov29.github.io/e-plantShopping/

## ✨ Funcionalidades

- Catálogo de plantas con imagen, nombre y precio.
- Botón **Add to Cart** en cada planta.
- Contador del carrito en el header que se actualiza al agregar productos.
- Página de carrito con:
  - Listado de plantas añadidas con imagen, nombre, precio unitario y cantidad.
  - Botones para incrementar y decrementar unidades por producto.
  - Cálculo automático del total de items y del costo total.
  - Botón de Checkout con mensaje tipo “Coming soon”.

## 🛠️ Tecnologías utilizadas

- React.
- JavaScript (ES6+).
- HTML5 y CSS3 para la estructura y estilos.
- Vite como herramienta de build y desarrollo local.
- GitHub Pages para el despliegue.

## ⚙️ Instalación y uso en local

Si quieres correr el proyecto en tu máquina de forma local:

1. Clona este repositorio:

   ```bash
   git clone https://github.com/edgardov29/e-plantShopping.git

   ```

2. Instala dependencias:

   ```bash
   npm install

   ```

3. Ejecuta en modo desarrollo:

   ```bash
   npm run dev

   ```

4. Para generar la versión de producción:

   ```bash
   npm run build
   ```
